# ذخیره کننده اینترنت 📱

یک اپلیکیشن Android برای ذخیره اینترنت شبانه و استفاده از آن بدون اینترنت

## ✨ ویژگی‌ها

- ✅ **ذخیره اینترنت شبانه** - تمام ترافیک را ذخیره کنید
- ✅ **پخش اینترنت ذخیره شده** - بدون اینترنت واقعی استفاده کنید
- ✅ **Proxy Server** - محلی و رایگان
- ✅ **VPN Compatible** - با VPN کار می‌کند
- ✅ **رابط ساده** - فقط 2 دکمه

## 🚀 نحوه استفاده

### مرحله 1: ذخیره اینترنت (شب 2-7 صبح)
1. اپلیکیشن را باز کنید
2. دکمه **"ذخیره اینترنت"** را بزنید
3. اینترنت رایگان را استفاده کنید
4. برنامه تمام ترافیک را ذخیره می‌کند

### مرحله 2: استفاده بدون اینترنت (روز بعد یا بعد)
1. اپلیکیشن را باز کنید
2. دکمه **"روشن کردن اینترنت ذخیره شده"** را بزنید
3. **اینترنت اصلی را خاموش کنید**
4. **VPN را روشن کنید** (اختیاری)
5. اکنون می‌توانید از اینترنت ذخیره شده استفاده کنید

## 📋 نیازمندی‌ها

- Android 5.0 (API Level 21) یا بالاتر
- 100 مگابایت فضای خالی
- اتصال اینترنت برای ذخیره

## 🔧 نصب

### از Android Studio:

1. **Android Studio را دانلود کنید**
   - https://developer.android.com/studio

2. **کد را Clone کنید**
   ```bash
   git clone https://github.com/fuuvgjkhfd-byte/nightmare-panel.git
   ```

3. **در Android Studio باز کنید**
   - File → Open → nightmare-panel

4. **Build و Run**
   - Run → Run 'app'
   - یا Ctrl+R (Windows) / Cmd+R (Mac)

5. **APK تولید کنید**
   - Build → Generate Signed Bundle / APK
   - اختیار "APK" را بزنید
   - تمام مراحل را تکمیل کنید
   - APK در `app/release/` ذخیره می‌شود

### از فایل APK:
1. APK را دانلود کنید
2. گوشی را روشن کنید
3. فایل APK را باز کنید
4. "نصب کنید" را بزنید

## 🐛 عیب یابی

**مشکل:** APK نصب نمی‌شود
- **راه‌حل:** منابع نامعلوم را فعال کنید
  - Settings → Security → Unknown Sources

**مشکل:** اینترنت ذخیره نمی‌شود
- **راه‌حل:** اتصال اینترنت را بررسی کنید

**مشکل:** پخش کار نمی‌کند
- **راه‌حل:** اینترنت اصلی را خاموش کنید
